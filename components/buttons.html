Buttons

This design system includes one primary button, one hover style, and one disabled button.


Button States


Normal

Hover

Disabled

Accessibility Considerations

Buttons are only used for performing an action on the current page (ex: form submissions). All three styles of buttons meet color contrast standards as verified using the WebAIM Contrast Checker. Proper HTML tags are used so that screen readers can differentiate between active, hover, and disabled button states.



Use Cases

  • Primary use is for navigating within a page.
  • Submission Forms
  • Will not be used for external links.
  • Contain text that is clear and concise.
