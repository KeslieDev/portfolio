Accessibility Projects

The navigation bar must be included on all pages created using this design system. The navigation bar includes a navigation menu and allows users to navigate to different pages of the website.


Responsive Example

Please feel free to resize this window to test out the responsiveness of the navigation bar.



Accessibility Considerations

Accessibility of the navigation bar was checked utilizing the Web Accessibility Evaluation Tool (WAVE) browser extension. During WAVE testing I found that my in-text links failed on a contrast check, so I increased the saturation to correct this issue. I also discovered that my LinkedIn and YouTube icons were missing the proper aria tag so that users with screen readers would know what the icons represented. I fixed this issue by inluding aria-label="alternative text" within the icon code.



Use Cases

  • Top-level menu navigation for all displays.
  • Responsive from mobile to large desktops.
  • Information Hierarchy is simple and easy to follow.
